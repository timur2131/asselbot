# Appointment Booking System (Aiogram)
Асинхронный Telegram-бот для автоматизации записи клиентов.

### Key Features:
- Архитектура на основе Router и State Machine (FSM).
- Интеграция с базой данных для хранения слотов и данных клиентов.
- Гибкая логика подтверждения записей.

### Tech Stack:
Python, Aiogram 3.x, PostgreSQL/Sqlite.
